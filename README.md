## GraphQL cheatsheet:  

### Queries: 

Basic query

{ status }    
↓   
{ status: 'available' }

Nesting  

{ hero { name height } }  
↓  
{ hero:  
    { name: "Luke Skywalker",   
      height: 1.74 } }
